Lung Cancer Risk Analysis Dashboard (Power BI)
Overview

This project presents an interactive Lung Cancer Risk Analysis Dashboard developed using Power BI.
It analyzes patient data to uncover relationships between lifestyle risk factors, symptoms, and lung cancer occurrence.

The dashboard enables users to explore patterns across age, gender, and health conditions through dynamic visualizations.

Key Objectives
Identify major risk factors contributing to lung cancer
Analyze symptom frequency among patients
Compare lung cancer distribution across demographics
Deliver insights through an interactive and user-friendly dashboard
🛠 Tools & Technologies
Power BI (Dashboard Development)
Power Query (Data Transformation)
DAX (Measures & Calculations)

Data Preparation
Cleaned and structured the dataset for analysis
Applied Unpivot transformation to normalize symptoms and risk factors
Created a Patient ID to ensure accurate aggregation
Used filtering techniques to analyze only relevant records (e.g., “Yes” values)

Dashboard 
 KPIs
Total Patients
Cancer Patients
Non-Cancer Patients
Average Age
Cancer Rate (%)

Visualizations
Lung Cancer Distribution
Cancer by Gender
Age Group Analysis
Smoking vs Lung Cancer
Top Symptoms (with conditional formatting)
Risk Factor Analysis

Interactivity
Slicers for Age, Gender, and Risk Factors
Dynamic filtering across all visuals

Key Insights
Smoking shows a strong association with lung cancer cases
Certain symptoms appear more frequently among diagnosed patients
Older age groups show higher cancer occurrence



Getting Started
